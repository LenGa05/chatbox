# Casanova Electrical Chat Assistant

A simple, rule-based chatbot built with **Flask** and **vanilla JavaScript**, designed to assist visitors of [Casanova Electrical](https://casanovaelectrical.com/) with quick answers, estimates, and emergency contact info.

---

## Features

- Lightweight **Flask web app** with responsive HTML/CSS/JS frontend  
- Smart **rule-based response system** (no AI model required)  
- Works both as a **web chat widget** and **command-line assistant**  
- Custom responses for estimates, outages, licensing, and services  
- Easy to deploy on your own domain or hosting provider
